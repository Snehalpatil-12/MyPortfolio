IoT Based Air Pollution Monitoring System

  • This project senses the air using gas sensor in a particular area and displays the percentage of pollution of that area 16x2 LCD
  • This project, IoT based air pollution monitoring system that constantly keeps track of air pollution in an area and displays the percentage of pollution on LCD screen it also sends data to “Thing speak” platform. This monitoring device can deliver real time measurements of air pollution.
  • IoT based air pollution monitoring system can be installed in industries, cities with heavy pollution, power plants etc.
